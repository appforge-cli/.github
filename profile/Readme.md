### [AppForge CLI](https://github.com/appforge-cli/appforge_cli)
A powerful command-line tool for generating Flutter apps with:
- 5 beautiful pre-built themes with light/dark mode
- Multiple auth methods (Email, Phone/OTP, Social)
- Modular Firebase integration (Auth, Firestore, Storage, FCM)
- 12+ production-ready reusable widgets
- Clean architecture with your choice of state management
- Responsive screens ready to customize

**Quick Start:**
```bash
dart pub global activate appforge_cli
appforge create
```
